# Seismic Hazard Detection using Machine Learning

## Overview
This project focuses on detecting and predicting seismic hazard levels using machine learning techniques. The objective is to analyze seismic event data, identify important patterns, and build predictive models that classify hazard severity.

The project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and visualization.

---

## Tech Stack
- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- NumPy
- Matplotlib / Seaborn

---

## Project Structure
- **database.csv** – Dataset containing seismic event records
- **ML_FINAL_PROJECT.ipynb** – Jupyter notebook with complete ML pipeline:
  - Data exploration and visualization  
  - Data preprocessing and cleaning  
  - Feature engineering  
  - Model training and evaluation  
  - Performance analysis and visualization  

---

## Features
- End-to-end machine learning pipeline implementation
- Exploratory Data Analysis (EDA) and visualization
- Feature engineering and preprocessing
- Model training using classification algorithms
- Model evaluation using accuracy and performance metrics

## Installation

1. Clone the repository:

```
git clone https://github.com/anshu1198/Seismic-Hazard-Mapping.git
cd Seismic-Hazard-Mapping
```

2. Install required packages:

```
pip install pandas numpy scikit-learn matplotlib jupyter
```

3. Open the Jupyter notebook:

```
jupyter notebook ML_FINAL_PROJECT.ipynb
```
